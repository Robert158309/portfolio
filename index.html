Proyectos

Sistema de Riego "SeedLive"

Consiste en un sistema de riego automatizado con capacidad de que le sean programados horarios y con gestion de distintos usuarios para la interaccion con el programa. Incluye Base de datos para el monitoreo de horarios y de inventario. Tambien tiene integración con módulos Arduino UNO. Fue presentado al público general durante la Expotec 2024 en el Instituto Emiliani Somascos.

Tecnologías: C#, DB2, Arduino

Sistema Modular en C++

Proyecto desarrollado en C++ con el objetivo de reforzar y mantener mis conocimientos en el lenguaje. Incluye diferentes módulos como calculadora, gestión de parqueo, librería y conversión de unidades, organizados en múltiples archivos para practicar una mejor estructura del código y el uso de funciones.

Tecnologías: C++, CMake
